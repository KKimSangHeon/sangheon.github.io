String

String 인스턴스 안에는 char 배열이 존재하는것이다.
String은 toString를 오버라이딩 하여 안에 내용이 출력된다.
toString를 오버라이딩 하여 인스턴스에 대한 정보를 확인할 수 있게 만드는 것은 좋은 습관이다. 디버깅할때 편하다는 장점이 있다.
