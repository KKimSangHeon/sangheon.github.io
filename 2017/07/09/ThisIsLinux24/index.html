2017-07-09
Linux, This Is Linux

네임서배 개요

네임서버=DNS(Domain Name System)서버
도메인 이름을 IP주소로 변환시켜 주는 역할 = 이름해석

네임서버의 위치를 저장해 놓는곳은?

실습

nate.com 접속방식

웹브라우저에서 www.nate.com을 치면 /etc/hosts에 접근하여 해당 ip가 있는지 확인을 한다. 존재 할 경우 그 ip로 접속하고 존재하지 않을 경우 /etc/resolv.conf 로 가서 네임서버를 확인하고 해당 ip주소를 알아온다.
